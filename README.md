# AI-generated-Text-Detector
This project detects whether a given text is AI-generated or human-written from a given pdf using Machine Learning.

## Models Used
-ANN
-Bidirectional LSTM

##Features
-PDF input support
-Text Preprocessing
-Duel Model prediction
- Confidence-baased decision

##Technologies
-Python
-TensorFlow/Keras
-Scikit-learn
-PyPDF2

##How it works
1.Upload a PDF
2.System extracts the texts
3.Models predict the result

##Output
-AI generated or Human Written with a Confidence Score
